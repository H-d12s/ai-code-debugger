# 🧠 AI Code Debugger

An AI-powered tool that analyzes code, explains errors, and provides fixes using Google's Gemini API.

## 🚀 Features
- Detects syntax and logic errors
- Explains bugs in simple terms
- Provides root cause analysis
- Suggests corrected code
- Severity rating (1–10)

## 🛠 Tech Stack
- Python
- Streamlit
- Gemini API

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
